## Technical Stuff

* The source is managed in git.
* The website is statically built using hugo.

To work on the website:

* Install hugo
* Install git lfs and pull git large files
* Run hugo:

  `hugo serve`

### Deployment

We deploy it using gitlab pipelines -- see .gitlab-ci.yml
